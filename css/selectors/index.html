Selectors

Type Selectors :

Style's the element using element name

Universal Selectors

Style gets applied to all the elements in the document

Class Selectors

style gets applied to elements have the same class name
  • Mutiple elements can have same class
  • Can you one more classes

ID Selectors

style gets applied to elements having that particular id

Note
  • ID should be unique to each element
  • Mutiple ID's are not allowed

Attribute Selectors

styling the element using element attributes
